# Fabiola — Portfolio

Personal website and portfolio for computational biology consulting.

Live site: https://fblbleng.github.io

## Structure

- `index.html` — homepage
- `css/style.css` — all styling
- `images/` — figures and photos
- `case-studies/` — detailed project case studies

## Local preview

Open `index.html` in any browser, or use RStudio's file pane to right-click → "View in Browser".

## Deploy

Commit and push to the `main` branch — GitHub Pages serves it automatically.
